# Windows Sandbox（サンドボックス）利用ガイド

## はじめに：Windows Sandboxってなに？

「Windows Sandbox（ウィンドウズ・サンドボックス）」は、あなたのパソコンの中にもう一つ、**「使い捨ての真っさらなWindows」** を立ち上げる機能です。

名前の由来は「砂場（Sandbox）」。子供が砂場で何を作っても、遊び終われば平らな砂に戻るように、この機能も**使い終わったら完全に元通り**になります。

### イメージしてください

* **いつものパソコン**: 大事な書類や設定が入っている「本番環境」。汚したくないリビングのような場所。
* **Windows Sandbox**: 何をしても許される「実験室」。実験が終われば、ボタン一つで部屋ごと爆破してキレイさっぱり消滅させることができます。

## どんなときに使うの？（メリット）

### 1. 怪しいファイルを安全に開きたいとき

「この添付ファイル、開いても大丈夫かな？」「このソフト、便利そうだけどウイルスじゃないかな？」と不安になったとき。
Sandboxの中で開けば、もしウイルスが入っていても、**あなたの本当のパソコン（リビング）には一切被害が及びません。**

### 2. その場限りのソフトを使いたいとき

授業や研修で「今日だけ使うフリーソフト」をインストールしたいとき。
いつものパソコンに入れると、パソコンが重くなったりゴミファイルが残ったりしますが、Sandboxなら**使い終わったら閉じるだけ**で、インストールした事実ごと消し去れます。

### 3. 設定変更の練習をしたいとき

「この設定を変えたらパソコンがおかしくなるかも…」という操作を試すとき。
Sandboxで練習して、失敗しても閉じてしまえば問題ありません。何度でもやり直しができます。

## 注意点（ここだけは覚えて！）

### 【重要】閉じると全部消えます！

これが最大の特徴であり、最大の注意点です。
Windows Sandboxの「×」ボタンを押して画面を閉じると、**その中で作ったファイル、保存したデータ、インストールしたソフトは、すべて「警告なし」で永久に消滅します。**

* **保存したいデータがある場合**: 閉じる前に、いつものパソコン側へコピー＆ペーストで救出してください。

## 使い方

### 起動方法

1. 画面下のスタートボタン（Windowsマーク）を押します。

2. 検索ボックスに「Sandbox」と入力します。

3. **「Windows Sandbox」** というアイコンが出てくるのでクリックします。

4. 少し待つと、青い画面の新しいウィンドウが開きます。これが「実験室」です。

### データのやり取り（コピー＆ペースト）

Sandboxは隔離されていますが、**コピー＆ペースト** だけは通じるようになっています。

1. **いつものパソコン**で、Sandboxに入れたいファイルを右クリックして「コピー」。

2. **Sandboxの画面内**で、デスクトップなどを右クリックして「貼り付け」。

3. これで実験室にファイルが持ち込まれます。逆（持ち出し）も可能です。

### 終了方法

ウィンドウ右上の「×」ボタンを押します。
「Windows サンドボックスを閉じると、すべてのコンテンツが破棄され完全に失われます。」というメッセージが出ます。「OK」を押すと、跡形もなく終了します。

## Q&A

**Q. Sandboxの中でインターネットは使えますか？**
A. はい、使えます。Edgeブラウザが入っているので、そこからファイルをダウンロードしたり検索したりできます。

**Q. 私のパソコンに入っていないのですが…**
A. Windows Sandboxを使うには、Windows 10 または 11 の **「Pro」** 以上のエディションが必要です。「Home」エディションでは標準で使えないため、表示されないことがあります。また、会社のパソコンなどでは管理者によって制限されている場合があります。

**Q. Sandboxを開いたら、必要なソフトが入っていません。**
A. はい、Sandboxは毎回「工場出荷状態（初期状態）」で始まります。WordもExcelも入っていません。
授業で特定の環境が必要な場合は、**専用の「起動用ファイル（ランチャー）」** を使うことで、起動と同時に必要なソフトを自動準備することができます。（※今回作成したツールがこれにあたります）
