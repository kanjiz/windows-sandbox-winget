# Winget（ウィンゲット）利用ガイド

## はじめに：Wingetってなに？

「Winget（ウィンゲット）」は、**魔法の呪文（コマンド）を唱えるだけで、ソフトを全自動でインストールしてくれる道具**です。

普通、パソコンにソフトを入れるときは、ブラウザで検索して、公式サイトを探して、ダウンロードして、ファイルをダブルクリックして、「次へ」「同意する」「次へ」…と何度もクリックしますよね。
Wingetを使うと、これらすべてを省略して、**たった一行の命令だけでインストールが完了**します。

### イメージしてください

- **いつもの方法**: 広いショッピングモールを歩き回って、目当ての商品を探し、レジに並んで買う（手間がかかる）。
- **Winget**: Amazonで欲しいものを検索して「ポチッ」と押すだけ。あとは勝手に届く（超高速）。

## なぜこれを使うの？（メリット）

### 1. 「次へ」連打からの解放

インストーラーの画面で何度も「次へ」を押す必要がありません。コマンドを一回打てば、あとはコーヒーを飲んでいる間に終わっています。

### 2. 安全なソフトだけが入る

Wingetはマイクロソフトが管理しているリストからソフトを探してきます。「検索して間違って偽物のサイトから変なウイルス入りソフトをダウンロードしてしまった…」という事故が防げます。

### 3. 一括インストールができる

パソコンを初期化したときや、今回のようなSandbox（実験室）環境で、「いつものソフト5個」を一つずつ入れるのは大変です。Wingetなら、それらを連続して命令できます。

## 使い方：基本の3ステップ

Wingetは、マウスではなくキーボードで文字を打って操作します。

### ステップ1：黒い画面（ターミナル）を開く

一番手っ取り早い「プロっぽい」方法を使いましょう。

1. キーボードの 「Windowsキー（窓のマーク）」を押しながら「X」 を押します。

1. メニューが出てくるので、「A」 を押します。（または、スタートボタンを右クリックして「ターミナル (管理者)」や「Windows PowerShell (管理者)」を選んでも同じです）

1. 「変更を加えることを許可しますか？」と出たら「はい」を選びます。

これで、黒や青の画面が開きます。ハッカー映画みたいに見えますが、ただの「命令受付窓口」なので怖がらなくて大丈夫です。

### ステップ2：欲しいソフトを探す（search）

まずは、ソフトの「正式名称」や「ID」を調べます。英語で `winget search` （ウィンゲット サーチ）と打ち、スペースを空けてソフト名を入力します。

**例：Google Chrome を探したいとき**

```powershell
winget search Chrome
```

これを入力して Enter キーを押すと、ズラッとリストが出てきます。
一番左の「名前」と、その隣にある **「ID（アイディー）」** に注目してください。
（例：`Google.Chrome` というIDが見つかります）

### ステップ3：インストールする（install）

IDがわかったら、`winget install` （ウィンゲット インストール）と打ち、スペースを空けてIDを入力します。

**例：Google Chrome を入れたいとき**

```powershell
winget install Google.Chrome
```

Enter キーを押すと、自動的にダウンロードとインストールが始まり、しばらくすると「インストールが完了しました」と表示されます。これだけで終わりです！

## よく使うコマンド一覧（カンニングペーパー）

これだけ覚えておけば大丈夫です。

| やりたいこと | 入力するコマンド（呪文） | 例 |
| :--- | :--- | :--- |
| **探す** | `winget search [キーワード]` | `winget search git` |
| **入れる** | `winget install [ID]` | `winget install Git.Git` |
| **確認する** | `winget list` | `winget list` |
| **全部更新** | `winget upgrade --all` | `winget upgrade --all` |

## ちょっとしたコツ

### 「ID」を使うのがオススメ

`winget install Chrome` のように「名前」でもインストールできますが、似たような名前のソフトが他にあると「どれのことですか？」と聞かれてエラーになることがあります。
`Google.Chrome` のように「ID」を指定すると、確実にそのソフトを一発で指名できます。

### 途中で止まったら？

たまに「同意しますか？ [Y] はい [N] いいえ」と聞かれることがあります。
そのときは、キーボードの `Y` を押して Enter を押してください。

## まとめ

Wingetは、**「ソフト名を入力するだけで、勝手にインストールしてくれる便利な執事」** です。
最初は黒い画面に抵抗があるかもしれませんが、慣れると「検索してダウンロードして…」という作業には戻れなくなるほど快適です。ぜひSandbox環境で練習してみてください。
